# RaspiNodeCamUSB

To use
Clone the repo into raspberry pi and connect a camera

inside raspberry do
sudo apt-get install fswebcam

1. npm install
2. npm install q

To Start app

node index.js 

The app should run in port 3000 of raspberry

To to take a pic 
Hit the URL 
http://rapiIP:3000/camera

This will return the image from camera 

Cheers !
